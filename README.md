# 📚 Word Formation Trainer

A simple and effective tool to practice English word formation

🌐 **Try it here:**
[jaduruch.github.io/word_formation_trainer](https://jaduruch.github.io/word_formation_trainer/)

---

## 🚀 Features

* Practice common **word families** (noun, adjective, adverb)
* Focus on **real exam-style vocabulary**
* Fast and easy to use — no signup required

---

## 🎯 Who is this for?

* Vocational high school students
* English learners (B1/B2)

## 🧠 What you’ll learn

* How to transform words into:

  * **Nouns** (e.g. *decide → decision*)
  * **Adjectives** (e.g. *care → careful*)
  * **Adverbs** (e.g. *quick → quickly*)

---

## 🛠️ Tech Stack

* HTML / CSS / JavaScript
* Hosted with **GitHub Pages**
